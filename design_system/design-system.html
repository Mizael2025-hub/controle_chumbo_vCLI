Grade 2D (estoque)

Toque uma célula p/ detalhes (não hover). Pinça p/ zoom. Célula nunca < 40px.

Em < 400px o modo compacto mostra 1 coluna (lista por posição).

Breakpoints (mobile-first, §7.4)

xs 360–430px — alvo principal. Bottom-bar, grade c/ scroll.
sm 431–767px — mesmo xs c/ respiro.
md 768–1023px — sidebar colapsável.
lg 1024px+ — sidebar fixa, layout completo.
xl 1280px+ — painéis laterais.

Regras anti-regressão relevantes (§6.9)

ARC-14 grade min-cell 40px, compacto < 400px · ARC-15 sem truncar ligas · ARC-16 texto curto sem siglas · ARC-17 FAB respeita safe-area.
